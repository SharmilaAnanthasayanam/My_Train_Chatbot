# My_Train_Chatbot

### ABOUT:
An AI-powered chatbot that facilitates users in accessing Indian train time table information

### Tools Used: 
* Streamlit
* Mongodb Atlas
* MySQL
* Pandas
* Sentence Transformer
* SKlearn for Cosine Similarity
* mysql-connector-python
